# Directory Overview

This directory is a collection of write-ups for Capture The Flag (CTF) challenges. Each markdown file documents the solution to a specific challenge, serving as a personal knowledge base and log of solved challenges.

# Key Files

The core content of this repository is the collection of markdown files, each detailing a CTF solution. The main files are:

*   `corrupted-file.md`: Documents how to repair a corrupted file by correcting its hexadecimal header.
*   `DISKO.md`: Explains how to extract readable strings, including flags, from a disk image using the `strings` utility.
*   `flag in the flame.md`: A write-up on decoding a large base64 encoded log file to reveal a hidden image containing the flag.
*   `Riddle Registry.md`: Details the process of finding a flag hidden in the metadata of a PDF or image file, using tools like `pdfinfo` and `exiftool`.

# Usage

This directory is intended for personal reference and learning. The user can add new markdown files to document solutions to new CTF challenges as they are solved.
